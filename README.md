# Sistema de Préstamo de Equipos para Laboratorio

## Descripción
Sistema web para gestionar préstamos de equipos de laboratorio con roles diferenciados:
- **Administrador**: CRUD completo de equipos y alumnos.
- **Alumno**: Solicita préstamos y registra devoluciones.
- **Consulta**: Solo lectura de alumnos e historial.

Funcionalidades extras:
- Cálculo automático de multas por retraso.
- Diseño responsive y moderno con Bootstrap 5.
- Desplegado en la nube (Render.com).

## Tecnologías
- Backend: Flask (Python)
- Frontend: Bootstrap 5, Bootstrap Icons
- Base de datos: SQLite (con SQLAlchemy)
- Despliegue: Render.com
